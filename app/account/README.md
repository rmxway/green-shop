# Account Module

Модуль личного кабинета пользователя с возможностью редактирования профиля.

## Структура

```
app/account/
├── components/          # UI компоненты
│   ├── ProfileView.tsx       # Отображение данных профиля
│   ├── ProfileEditForm.tsx   # Форма редактирования
│   └── ProfileActions.tsx    # Кнопки действий
├── hooks/              # Бизнес-логика
│   └── useProfileEdit.ts     # Логика редактирования профиля
├── orders/             # Страница заказов
├── content.tsx         # Главный компонент страницы
├── styled.ts           # Styled-компоненты
└── page.tsx            # Next.js page
```

## Компоненты

### `ProfileView`
Отображает статичные данные профиля (имя, фамилия, телефон, адрес).

### `ProfileEditForm`
Форма с инпутами для редактирования профиля.

### `ProfileActions`
Управляющие кнопки: Редактировать/Отмена/Сохранить, Мои заказы, Выйти.

## Хук `useProfileEdit`

Инкапсулирует всю логику работы с профилем:
- Управление формой и режимом редактирования
- Сохранение данных на сервер
- Обновление сессии и локального snapshot для мгновенного обновления UI
- Обработка ошибок

## Принципы

- **DRY**: повторяющиеся поля вынесены в массивы конфигурации
- **KISS**: логика отделена от представления
- **SRP**: каждый компонент отвечает за одну задачу
